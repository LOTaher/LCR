# LCR

> Laith's Container Registry

The official webpage can be found at https://registry.splatte.dev

LCR is my personal container registry to upload images to and from. It is hosted within my [homelab](https://github.com/LOTaher/LIONS), and is what I use for personal containerized development.

Currently, the project is not OCI compliant. It will only work with [Docker](https://www.docker.com/).

> But why not use [Distribution](https://github.com/distribution/distribution/tree/main)?

I want to learn how the technology works, so I am writing it by hand. This is an implementation is also meant to work exactly how I'd like in my homelab.

This project is not currently structured or documented for public use. Though, it is published for transparency and source availability under the terms of the GPL.
